Create a new partner organization account

Click "Generate" to create a database-verified unique code

Note: Partner codes start with "T4LP-" to distinguish them from ambassador codes

Click "Generate" to create a secure random password

Partners are typically created as "Approved" by default

Partner vs Ambassador

  • • Ambassadors: Use codes starting with "T4LA-"
  • • Partners: Use codes starting with "T4LP-"
  • • Partners can post opportunities for ambassadors
  • • Default password is "welcome123" for both
